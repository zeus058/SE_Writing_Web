# Requirements Analysis (Phân tích Yêu cầu)

Thư mục này chứa các tài liệu liên quan đến việc khảo sát, thu thập và phân tích yêu cầu của dự án **YAG - Writing Novels Web**.

## Các nội dung chính
- **Yêu cầu chức năng (Functional Requirements):** Danh sách các tính năng hệ thống phục vụ Độc giả, Tác giả và Quản trị viên.
- **Yêu cầu phi chức năng (Non-functional Requirements):** Các tiêu chuẩn về bảo mật, hiệu năng, khả năng mở rộng, độ ổn định và tính tương thích.
- **Use Case Diagrams & Specifications:** Sơ đồ ca sử dụng chi tiết cho từng phân hệ chính (Viết truyện, Diễn đàn, Tìm kiếm AI, Thanh toán, ...).
- **Stakeholders Analysis:** Phân tích các bên liên quan trực tiếp và gián tiếp đến hệ thống.
