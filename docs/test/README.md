# System Testing (Kiểm thử Hệ thống)

Thư mục này chứa các tài liệu, kịch bản kiểm thử (Testcases) và báo cáo kết quả kiểm thử cho dự án **YAG - Writing Novels Web**.

## Các nội dung chính
- **Kế hoạch kiểm thử (Test Plan):** Định nghĩa phạm vi kiểm thử, chiến lược kiểm thử, tài nguyên và lịch trình thực hiện.
- **Thiết kế ca kiểm thử (Testcase Design):**
  - **Kiểm thử chức năng (Functional Testing):** Các kịch bản kiểm thử cho Xác thực, Sáng tác, Thanh toán VNPAY, Diễn đàn, Tìm kiếm AI.
  - **Kiểm thử phi chức năng (Non-Functional Testing):** Kiểm thử hiệu năng (Tải trọng tìm kiếm AI, độ trễ WebSocket), kiểm thử bảo mật (chữ ký số VNPAY, Bcrypt, Brute-force).
- **Báo cáo kết quả kiểm thử (Test Report):** Thống kê tỷ lệ đạt/lỗi (Pass/Fail rate), các lỗi phát hiện và lịch sử sửa lỗi.
